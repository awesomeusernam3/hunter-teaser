## What is Old School RuneScape & RuneLite?

**[Old School RuneScape (OSRS)](https://runescape.com)** is a massively multiplayer online role-playing game (MMORPG) released by Jagex. It is based on a snapshot of the game from 2007, blending retro mechanics and high-difficulty gameplay with active, community-driven content updates.

Because the game features vast mathematical complexities, tick-based combat rules, and limited default user interfaces, the player base heavily relies on **[RuneLite](https://runelite.net)**. RuneLite is an open-source, community-led client written in Java that injects visual overlays, event tracking, and quality-of-life adjustments directly over the official game stream.

**This repository hosts a modular Java-based plugin built specifically for the RuneLite client.**

---

## About The Plugin

This plugin enhances <a href="https://oldschool.runescape.wiki/w/Pitfall">pitfall hunting</a> by making your character verbally taunt the creature whenever you poke it, adding immersion, attitude, and a bit of chaos to the hunt.

## Local Development Setup

Follow these steps to run a local development instance of RuneLite containing your custom plugin code.

### Prerequisites
* **Java Development Kit (JDK) 11 or 17** (Match the current target of the RuneLite API).
* **IntelliJ IDEA** (Highly recommended for RuneLite development).
* **Lombok Plugin** installed in your IDE.

### Installation & Execution

See <a href="https://github.com/runelite/plugin-hub">Runelite's plugin-hub</a> repository for more information.
